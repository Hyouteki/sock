Simple expression transformer inspired by [Coq](https://github.com/coq/coq) and [Noq](https://github.com/tsoding/Noq). 

> **Important**
> For more information refer [wiki](https://github.com/Hyouteki/sock/wiki)

## Quick Start
``` console
make build
make run
```
